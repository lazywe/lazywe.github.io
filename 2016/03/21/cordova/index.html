注意

cordova ionic 安装

  • 安装ionic cordova (可单独安装)
  • mac下请使用超级身份 sudo

创建应用

  • 命令详解： ionic start 项目名称 模版名称
  • ionic官方提供了三套模版
  • blank（空白模版） tabs（底部导航模版） sidemenu（左侧导航模版）
