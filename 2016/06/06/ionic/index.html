    JAVA环境

    • 下载jdk http://jdk.android-studio.org/
    • 下载sdk http://sdk.android-studio.org/
    • 根据自己的操作系统下载自己的版本

    JAVA环境变量

    • 变量名 ：JAVA_HOME,变量值：当前jdk安装目录
    • 变量名 ：CLASSPATH,变量值：JAVA_HOME下的lib目录
    • 变量名 ：PATH,变量值： JAVA_HOME下的bin目录，以及JAVA_HOME下jre的bin目录

    配置方法

    window
    • 通过我的电脑右键－> 属性－>环境变量设置
    mac
    • which java
    • /usr/libexec/java_home 定位当前javahome路径
    • 生效环境变量
    • 环境测试

    SDK环境变量

    • 变量名 ：ANDROID_HOME,变量值：当前sdk安装目录

    • 变量名 ：PATH,变量值：ANDROID_HOME下的tools目录,ANDROID_HOME下的platform-tools目录

    • 打开 android sdk manager

    • 安装
    • Tools下所有
    • Extras下的inter x86 emulator…
    • Android（Api23,Api21,Api20…）根据自己的需求安装 我只按了个Api20
    新建模拟器
    • 点击菜单栏的 Tools的 Manage AVDs… 创建模拟器
    • 详细请 点击查看
    注意事项
    • mac系统 模拟器cpu使用inter时 需要下载HAXM
    • 下载地址 点击查看

    Cordova模拟器测试，真机调试

    • 真机调试比较简单,只要确保你的模拟器可以正常启动，java环境，android环境配置成功就可以跑起来
    • 第一步 添加cordova安卓平台
    • 第二步，模拟器运行,你需要吧emulator启动
    • 第三步，真机运行 将手机链接上电脑并且启动手机的调试
    • 第四步 ok 大功告成 附上一些资源地址
