    • 简介

    • RaisedButton控件一个凸起的材质矩形按钮

    属性分析

    onPressed

    • {() → void onPressed}
    • 按钮点击事件， 若存在按钮enable，不存在按钮disable

    onHighlightChanged

    • {(bool) → void onHighlightChanged}
    • 改变高亮颜色回掉函数，一个按钮会触发两次，按下后改变时触发一次，松手后恢复原始颜色触发一次
    • 参数 bool，按下后true，恢复false

    textTheme

    • ButtonTextTheme
    • 文本颜色模版，可根据系统模版，自动渲染按钮
    • 可选值：ButtonTextTheme.accent，使用模版颜色的。ThemeData.accentColor.
    • 可选值：ButtonTextTheme.normal，按钮文本是黑色或白色取决于。ThemeData.brightness.
    • 可选值：ButtonTextTheme.primary，按钮文本基于。ThemeData.primaryColor.

    textColor

    • Color
    • 文本颜色
    • eg：Colors.red 红色

    disabledTextColor

    • Color
    • 禁用按钮文本颜色

    color

    • Color
    • 按钮背景颜色

    disabledColor

    • Color
    • 禁用按钮的背景颜色

    highlightColor

    • Color
    • 高亮背景颜色

    splashColor

    • Color
    • 飞溅颜色
    • 场景，按钮enable状态，触碰区域的颜色

    colorBrightness

    • Brightness
    • 颜色亮度

    elevation

    • double
    • enable阴影高度

    highlightElevation

    • double
    • 按钮按下后阴影高度

    disabledElevation

    • double
    • 按钮禁用状态下阴影高度

    padding

    • EdgeInsets
    • 内边距
    • eg: EdgeInsets.only(left: 6.0, top: 3.0, right: 3.0, bottom: 6.0)

    shape

    • ShapeBorder
    • 边框
    • eg new Border.all(color: Colors.red, width: 2.0, style: BorderStyle.solid)
